# ✋ Right, Listen Up

One thing. Right now. For when the planning has become the procrastination.

---

## What it does

Reads the conversation history. If you've mentioned the same task, idea, or problem more than twice without meaningful progress — if the questions have become more elaborate while the action has stayed at zero — it calls it out. Directly, warmly, and without ceremony.

Then it gives you one thing. Not a plan. Not a roadmap. One stupidly small first step you can do right now. Then it stops talking, which is more than most productivity tools manage.

If you push back and say you're actually thinking, it accepts that gracefully and helps you think. It's not precious about being right.

---

## When it fires

**Fires when:**
- The same task or problem has come up more than twice with no action taken
- The questions are getting more elaborate while the doing stays at zero
- You're visibly in planning-as-avoidance mode
- You say things like "I keep meaning to..." or "once I've sorted X I'll..."

**Does not fire when:**
- The problem genuinely requires planning and research
- It's a legitimate multi-stage problem
- There's any real doubt about whether it's avoidance versus thinking

---

## Quick example

> **You:** *(Third conversation this week about the same side project that hasn't started)*
> So I'm thinking I should probably figure out the architecture first, then the data model, then maybe prototype the core loop before I touch the UI...
>
> **Claude:** Right, listen up.
>
> You've been "about to start" this project for three conversations now. The architecture is not the blocker. The starting is the blocker.
>
> One thing: open a new folder. Name it. That's it. Do that now.
>
> **[I know, you're right — what's the one thing?]** / **[I'm not procrastinating, I'm thinking]**
