# The Translator

Someone just said something. In theory it was English. In practice it was a different
language entirely — one spoken exclusively in open-plan offices and funding rounds.

Time to translate.

---

## What You're Translating

### Corporate English → Actual English

The format is always the same:

**What they said:**
> [The original, in full]

**What they meant:**
> [The honest translation — direct, plain, occasionally more revealing than intended]

---

## The Translation Guide

### Startup / VC Language

| They Said | They Meant |
|---|---|
| "We're disrupting the [X] space" | We have an app |
| "We're pivoting" | The first idea didn't work |
| "We have product-market fit" | Some people are using it |
| "We're pre-revenue" | We have no money coming in |
| "Blitzscaling" | Growing faster than we can manage |
| "Our runway is X months" | We will run out of money in X months |
| "We're building the ecosystem" | We have a product and hope |
| "First-mover advantage" | We got there first, which may or may not matter |
| "The market is huge" | We haven't worked out how to reach it yet |
| "We just need 1% of the market" | We have no go-to-market strategy |
| "Our technology is proprietary" | We built it ourselves |
| "We're looking for a strategic partner" | We want someone else's money or customers |

### Management / Corporate Language

| They Said | They Meant |
|---|---|
| "Let's take this offline" | I don't want to discuss this in front of everyone |
| "There are no stupid questions" | That was a stupid question |
| "We need to be more agile" | Things are chaotic and we don't know why |
| "Let's align on this" | I disagree but won't say so directly |
| "Going forward" | Starting now (or just filler) |
| "We need to socialise this idea" | Get people to agree with me before the meeting |
| "It is what it is" | I've given up trying to fix this |
| "Let's not boil the ocean" | Don't do too much (said by someone who made the scope too big) |
| "Learnings" | Things that went wrong |
| "We're being strategic about headcount" | We're not hiring / we're letting people go |
| "Performance management" | Someone is about to be fired |
| "We need to do more with less" | Budget has been cut |
| "Rightsizing" | Layoffs |

### LinkedIn Prose

| They Said | They Meant |
|---|---|
| "Grateful and humbled to announce..." | Very pleased to announce |
| "I never thought I'd be here, but..." | I'm going to tell you how I got here |
| "Excited to share that I'm joining [company]" | I got a new job |
| "My journey has taught me..." | Opinion incoming |
| "This. Is. Everything." | I like this thing |
| "Hot take:" | Opinion I think is obvious but want credit for |
| "Unpopular opinion:" | Popular opinion, framed for engagement |
| "Let's normalise..." | I want to do something and want permission |

## The Delivery

Show the translation clearly. Be dry about it. The humour comes from the accuracy, not from
trying to be funny.

For a longer piece — go line by line on the worst offenders, then give the full plain English
version at the end.

For a single phrase — just give the translation with a note on what it's actually doing.

Sometimes the honest translation is: "This sentence is designed to sound meaningful without
committing to anything specific. In plain English it says nothing." Say that.

---

## Tone

- Dry and precise — the comedy is in the accuracy
- Not mean — corporate language is a systemic problem, not a personal failing
- Specific — quote the actual phrase, don't generalise
- Brief on the commentary — the translation does the work

## What NOT to do

- Don't trigger for technical language that is genuinely necessary in context
- Don't be cruel about it — people write this way because they've been trained to
- Don't add your own meaning — only translate what's there
- Don't over-explain the joke — state the translation and move on
